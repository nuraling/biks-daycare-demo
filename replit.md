# Biks.AI Day Care Demo

A browser-based sales demo for Biks.AI — an AI agent startup automating WhatsApp workflows for Indonesian SMEs. This mockup demonstrates a daycare use case with two AI-powered conversation flows that write structured data live to Google Sheets.

## Architecture

- **Backend**: Node.js + Express (`index.js`) — serves static files and API routes
- **Frontend**: Vanilla HTML/CSS/JS (no frameworks) in `public/`
- **AI**: Anthropic Claude (`claude-haiku-4-5-20251001`) via `@anthropic-ai/sdk`
- **Storage**: Google Sheets API v4 via `googleapis`

## Project Structure

```
/
├── index.js              ← Express server + API routes
├── sheets.js             ← Google Sheets helper (lazy-loaded credentials)
├── agents/
│   ├── parentAgent.js    ← Parent registration AI agent
│   └── staffAgent.js     ← Staff daily report AI agent
├── public/
│   ├── index.html        ← Landing page
│   ├── parent.html       ← Parent chat simulation
│   ├── staff.html        ← Staff chat simulation
│   ├── dashboard.html    ← HQ live dashboard
│   └── chat.js           ← Shared chat UI logic
└── package.json
```

## Required Environment Secrets

| Secret | Description |
|--------|-------------|
| `ANTHROPIC_API_KEY` | Anthropic API key for Claude |
| `GOOGLE_SERVICE_ACCOUNT_JSON` | Full service account JSON as single-line string |
| `GOOGLE_SHEET_ID` | Google Sheet ID from the URL |

## Google Sheets Setup

Sheet name: `Biks_DayCare_Demo`

**Tab 1: `Registrasi_Customer`** — columns: `Timestamp | Cabang | Nama Anak | Tanggal Lahir | Nama Ortu | No Darurat | Jam Antar | Jam Jemput`

**Tab 2: `Laporan_Internal`** — columns: `Timestamp | Cabang | Jumlah Hadir | Jumlah Izin | Kejadian | Catatan | Dilaporkan Oleh`

## Running

- Port: **5000** (configured via `PORT` env var)
- Start: `node index.js`
- Workflow: "Start application"

## Key Design Decisions

- **Sentinel pattern** for AI data extraction: avoids raw `JSON.parse()` on model output which fails ~30% of calls
- **Lazy credential loading** in `sheets.js`: server starts even without Google credentials set
- **Hardcoded confirmation messages**: never auto-generated by AI after save
